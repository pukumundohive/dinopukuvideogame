DinoPuku Video Game - Information Page

How to Play the Game

  1. Click on the "Run" button at the top of this Replit workspace
  2. Wait for the game to launch (it might take a few seconds)
  3. The game will appear in the "Game Runner" workflow tab
  4. Click on the game window to give it focus and start playing!
Controls:
SPACE / UP ARROW - Jump
DOWN ARROW - Duck

Game Features

This game features two types of collectibles:

  • HBD Coins - Green coins that give 1 point
  • Hive Power - Red coins that give 5 points

The updated version has:

  • Separate counters for both coin types
  • Green-colored HBD counter
  • Improved image quality for start and end screens
  • Bigger airplane that appears more frequently
  • Sound disabled for better performance
